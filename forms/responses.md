## ユーザー傾向分析テンプレート（月次）

30件以上の回答が集まったら以下のテンプレートで分析する。

```
【フォーム回答 傾向分析】YYYY年MM月

■ 最多ユーザーゾーン
→ 雇用形態：XX / 年齢：XX歳代 / 悩み：XX

■ 転職温度感
→ 「できるだけ早く」：XX% / 「まだ決めていない」：XX%

■ Threadsコンテンツへの反映
→ 最多の悩み「XX」に対応した投稿を増やす
→ 参照：threads/generate_posts.md / knowledge/content_pillars.md

■ LINE対応への反映
→ 温度感が低い（情報収集段階が多い）場合：step_messages.md の教育フェーズを強化
→ 温度感が高い（早く転職したいが多い）場合：相談誘導を前倒しにする
→ 参照：line/step_messages.md / line/conversion.md

■ フォーム改善への反映
→ Q3「その他」が多い場合：選択肢を見直す
→ 参照：forms/improvement.md
```

## ユーザーセグメント別フォロー方針

Q1とQ4の組み合わせでLINEでの優先度を変える。

| セグメント | Q1 | Q4 | LINE対応方針 |
|---------|----|----|------------|
| 即戦力ユーザー | 第二新卒 | できるだけ早く | 48時間以内に相談日程を確定する |
| 悩み中ユーザー | フリーター | まだ決めていない | step_messages.md の教育フェーズを丁寧にこなす |
| 情報収集ユーザー | 既卒 | 半年〜1年以内 | broadcast.md の定期配信でロングフォローする |
| 高齢フリーター | フリーター | 30歳以上 | 「年齢的に今が大事」のメッセージを早めに伝える |
