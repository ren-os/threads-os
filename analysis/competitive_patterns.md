# competitive_patterns.md — 競合パターン週次分析

毎週月曜日に `research/winning_posts.md` の蓄積データを集計し、
最も伸びたフック・構成・CTA・テーマをランキング化する。
分析結果は `knowledge/post_patterns.md` の優先度に自動反映する。

---

## 実施タイミング・担当

```
タイミング：毎週月曜（automation/weekly.md タスク4 内）
所要時間：15〜20分
実施場所：このファイルに週次レポートを追記する
反映先：knowledge/post_patterns.md のパフォーマンスサマリー
```

---

## 週次分析の手順

```
Step1：research/winning_posts.md を開く
  → 今週記録した勝ち投稿を全件確認する
  → Type別バズカウンターを見て今週の傾向を把握する

Step2：4軸ランキングを作成する（下記テンプレートを使用）
  ① フックランキング（感情タイプ別）
  ② 構成ランキング（Type-A〜F）
  ③ CTAランキング（方法・フレーズ別）
  ④ テーマランキング（ジャンル別）

Step3：knowledge/post_patterns.md の優先度を更新する
  → 累計バズ観測数 3以上 → ★★★ に昇格
  → 累計バズ観測数 2      → ★★ に昇格
  → 自己投稿3本連続低パフォーマンス → 降格

Step4：threads/ideas.md に来週の投稿アイデアを追加する
  → 今週のトップType・トップテーマから転用案を作る
```
