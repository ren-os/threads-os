# content_analysis.md — コンテンツ多軸分析

Threads投稿をジャンル・型・時間帯・CTA・フック別に分析する。
viral_analysis.md の「集計・傾向分析版」。

## 分析の実施タイミング

- **月次：** 前月の全投稿を5軸で集計する（monthly_review.md と同タイミング）
