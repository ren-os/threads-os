# viral_analysis.md — バズ投稿・沈没投稿分析

伸びた投稿・伸びなかった投稿の要因を分析し、次回の投稿品質を上げる。
threads/viral_patterns.md・threads/posted_log.md と連動する。

---

## 分析の実施タイミング

- **週次：** 先週の全投稿を簡易チェック（いいね・保存・遷移が多いものをピックアップ）
- **月次：** TOP3・WORST3を深掘り分析し、パターンを言語化する

---

## 週次バズチェック（簡易）

毎週月曜日に実施。5分以内で完了させる。


## 要因分析フレームワーク

伸びた/伸びなかった理由を以下の軸で分析する。

| 分析軸 | 確認ポイント |
|-------|-----------|
| フック | 冒頭1〜2行で読者を引きつけているか |
| 共感 | ターゲット（第二新卒/フリーター）の気持ちを代弁しているか |
| 情報価値 | 読んで「ためになった」「保存したい」と感じる内容か |
| CTA | コメントや返信を誘発しているか |
| 時間帯 | 07:00/10:00/13:00/18:00/20:00 の最適時間帯に投稿しているか |
| 型 | T01〜T10のどの型で、その型がジャンルに合っているか |
| 文字数 | 150〜350字の範囲に収まっているか |
