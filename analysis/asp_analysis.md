# asp_analysis.md — ASP分析

案件別CVR・承認率・報酬を分析し、紹介戦略を最適化する。
asp/kpi.md・asp/tracking.md のデータをもとに深掘り分析する。

---

## 分析の実施タイミング

- **月次：** ASP管理画面確認後に分析を実施（毎月1日）
- **四半期：** 案件ポートフォリオを見直し、入れ替えを判断する
