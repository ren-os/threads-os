# blog_analysis.md — ブログ分析

PV・検索順位・CTR・滞在時間・CVRを分析し改善案を生成する。
blog/kpi.md のデータをもとに深掘り分析する。

---

## 分析の実施タイミング

- **週次：** 主要キーワードの順位変動を確認（5分）
- **月次：** 全指標を深掘りし改善案を生成（20分）

## 反映先ファイル

| 改善の方向性 | 反映先 |
|-----------|-------|
| 順位が下落した記事 | blog/update_rules.md のリライトフロー |
| CTRが低い記事 | タイトルを blog/seo.md のルールで改善 |
| 意外なクエリで流入している | blog/ideas.md に新規記事ネタとして追加 |
| LINE登録CVRが低い | blog/article_templates.md のCTAを見直す |
| 滞在時間が短い | 冒頭の構成・内部リンクを blog/internal_links.md で強化 |
