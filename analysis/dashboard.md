# dashboard.md — 全体KPIダッシュボード

Threads → LINE → フォーム → 相談 → ASP のファネル全体を1画面で把握する。
毎週月曜日に更新する。

---

## ファネル全体図

```
[Threads投稿]
    ↓ インプレッション → プロフィール遷移率
[プロフィール]
    ↓ LINE登録率
[LINE友達追加]
    ↓ フォーム回答率
[Googleフォーム]
    ↓ 相談予約率
[個別相談]
    ↓ エージェント登録率
[転職エージェント登録]
    ↓ ASP承認率
[ASP報酬確定]
```
