# checklist.md — 記事公開前チェックリスト

記事を公開する前に必ず確認するチェックリスト。
SEO品質・CV導線・法的要件を一括確認する。

---

## 使い方

記事を公開する前に、以下のチェックをすべて完了させる。
チェックが通らない項目は修正してから公開する。

---

## A. 基本品質チェック

```
□ タイトルは30〜35文字以内か
□ タイトルに検索キーワードが含まれているか
□ タイトルに数字・年・ターゲット属性が含まれているか（例：「2026年」「フリーター」）
□ 冒頭200字でターゲットの悩みに共感し、記事で解決できることを宣言しているか
□ 文字数は3,000字以上か
□ H2は4〜7個あるか
□ H2の一部に検索キーワードが含まれているか（自然な文脈で）
□ 読んでいて誤字・脱字がないか
□ 断言・誇張表現（「必ず」「100%」等）を使っていないか
```

---

## B. SEOチェック（seo.md参照）

```
□ メタタイトルが設定されているか（タイトルと同一でOK・120文字以内）
□ メタディスクリプションが設定されているか（120字以内・キーワード含む）
□ URLスラッグが英数字・ハイフン形式で設定されているか（例：freelance-to-seishain）
□ 画像のalt属性が設定されているか（すべての画像に）
□ 更新日・公開日が記事に表示されているか
□ パンくずリストが設定されているか
□ モバイル表示を確認したか（スマホで表示して崩れていないか）
□ ページの表示速度が遅くないか（画像サイズが大きすぎないか）
```

---

## C. 内部リンクチェック（internal_links.md参照）

```
□ 記事内に内部リンクが3本以上あるか
□ アンカーテキストが「こちら」「詳細」ではなく内容がわかる文言になっているか
□ カテゴリTOPページへのリンクが含まれているか
□ 関連記事ブロック（記事末尾）に2〜3本の記事リンクを設置したか
□ リンク先のURLが正しく、404エラーになっていないか
```

---

## D. CV導線チェック（affiliate_strategy.md / article_templates.md参照）

```
□ LINE誘導ブロックが本文中に2回以上入っているか
□ LINE誘導ブロックが記事末尾に入っているか
□ LINE登録リンクは正しく設定されているか（TODO：リンク確定後に確認）
□ アフィリエイトリンクが正しく設定されているか（エージェントレビュー・比較記事）
□ CTAの文章に「完全無料」「強引な勧誘なし」が含まれているか
□ 記事の意図（Know/Do/Go/Buy）に合ったCTAタイプを選んでいるか
```

---

## E. E-E-A-T・信頼性チェック（seo.md参照）

```
□ 記事内に具体的な体験・事例が含まれているか（「〇歳・〇ヶ月・〇円」等の数字）
□ 情報の出典（公式データ・厚労省等）を明記しているか（必要な場合）
□ プロフィールページへのリンクが記事内または記事近くにあるか（TODO：プロフィール作成後）
□ Threadsアカウント（@kaishain_nigemichi）へのリンクまたは言及があるか
□ 誇張・不正確な情報が含まれていないか
```

---

## F. 法的・規約チェック

```
□ アフィリエイトリンクを含む場合、記事冒頭に「この記事にはプロモーション（広告）が含まれています」と記載したか
□ 各ASPリンクの近くに「（PR）」表記があるか
□ 個人情報・特定個人が識別できる情報を含んでいないか
□ 他サイト・他記事からのコピー・無断引用がないか
```

---

## G. 公開後の初動確認（公開直後）

```
□ Googleサーチコンソールでインデックス登録をリクエストしたか
□ Twitterまたは他のSNSでシェアしたか（TODO：Threads運用と連動）
□ 関連する既存記事からこの記事へのリンクを追加したか
□ kpi.md の記事リストに公開日・タイトルを記録したか
□ ideas.md の「使用済みネタログ」にこの記事を移動したか
```

---

## チェックが通らない場合の対応

| チェック項目 | 修正例 |
|-----------|------|
| 文字数が3,000字未満 | H2を1〜2個追加し、FAQ・まとめセクションを充実させる |
| LINE誘導が1ヶ所以下 | 本文中にインラインCTAを追加する（article_templates.md参照） |
| PR表記がない | 記事冒頭と各ASPリンク近くに追記する |
| 内部リンクが2本以下 | 関連記事ブロックを追加する |
