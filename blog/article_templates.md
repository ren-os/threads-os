# article_templates.md — 記事テンプレート集

記事の型を定義する。型に沿って書くことで品質を安定させ、CVR（LINE登録率）を最大化する。

---

## 共通ルール（全テンプレート共通）

- 文字数：3,000〜6,000字（SEO最低ライン3,000字）
- LINE誘導ブロックは本文中に2回以上・記事末尾に必ず入れる
- 見出し（H2）は4〜7個
- タイトルに数字・年・ターゲット属性を入れる（例：「フリーター 25歳」「2026年最新」）
- 冒頭200字でターゲットの悩みに共感し、記事で解決できることを宣言する

---

## テンプレートT01：比較記事

**用途：** 転職エージェント・サービスを複数比較する記事
**検索意図：** 「〇〇 比較」「〇〇 おすすめ」
**CVRが高い理由：** 読者がサービス選定フェーズにいるため

### 構成テンプレート

```
[タイトル]
【2026年最新】第二新卒におすすめの転職エージェント〇社比較｜フリーター・既卒も対応

[冒頭（200字以内）]
この記事では、第二新卒・フリーター・既卒の方が使える転職エージェント〇社を
実際に使った視点から比較します。
「どのサービスを選べばいいかわからない」という方は最後まで読んでください。

H2：結論：この3つがおすすめ（まとめ表）
→ 比較表（エージェント名・対象・特徴・公式URL）を最初に出す
→ 忙しい読者のために結論ファースト

H2：比較の選び方・基準
→ 「なぜこの3社を選んだか」の根拠を書く
→ E-E-A-T対策（実際に使った、調査した）

H2：エージェントA の特徴・評判
→ 良い点3つ・気になる点2つ
→ こんな人に向いている

H2：エージェントB の特徴・評判
（同上）

H2：エージェントC の特徴・評判
（同上）

H2：あなたに合ったエージェントの選び方
→ タイプ別おすすめを整理（フリーター向け・第二新卒向け等）

[LINE誘導ブロック①：H2の後半に挿入]
---
「どれを選べばいいかわからない」という方へ
無料で個別相談を受け付けています。
あなたの状況に合ったエージェントを一緒に選びます。
👉 [LINE登録リンク] ← TODO
---

H2：よくある質問（FAQ）
→ 3〜5問のQ&A

[記事末尾のLINE誘導ブロック②]
---
この記事を読んでもまだ迷っていたら、LINEで直接相談してください。
第二新卒・フリーター・既卒の転職を無料でサポートしています。
👉 [LINE登録リンク] ← TODO
---
```

## テンプレートT02：ランキング記事

**用途：** 「おすすめ〇選」「人気ランキング」記事
**検索意図：** 「〇〇 おすすめ」「〇〇 ランキング」
**CVRが高い理由：** 「1位を使いたい」という心理が働く

### 構成テンプレート

```
[タイトル]
フリーターにおすすめの就職支援サービスTOP5【2026年最新】

[冒頭（200字以内）]
この記事では、フリーターが正社員を目指すために使える
就職支援サービスをランキング形式で紹介します。
「何から始めればいいかわからない」方向けに選定基準も解説します。

H2：選定基準（なぜこのランキングなのか）
→ 選定理由を明示（E-E-A-T対策）

H2：1位：〇〇
→ こんな人に向いている
→ 良い点・気になる点
→ 公式リンクとれんのひとことコメント

H2：2位：〇〇
（同上）

H2：3位以降
（同上・簡略版OK）

[LINE誘導ブロック①]

H2：ランキングの選び方（あなたに合ったサービスを見つける）
→ タイプ別診断（フリーター年数・年齢・希望職種）

H2：よくある質問（FAQ）

[LINE誘導ブロック②（記事末尾）]
```

## テンプレートT03：体験談記事

**用途：** 実際の相談事例・転職事例をストーリー形式で書く記事
**検索意図：** 「フリーター 転職 体験談」「既卒 正社員 なれた」
**CVRが高い理由：** 「自分と同じ状況の人が成功している」→行動意欲が上がる

### 構成テンプレート

```
[タイトル]
フリーター26歳が正社員に転職した体験談【3ヶ月で内定が出た方法】

[冒頭（200字以内）]
「フリーター歴5年、もう正社員は無理かも」と思っていた26歳の方が
転職相談を通じて3ヶ月で内定を獲得した話をご紹介します。

H2：転職前の状況（ビフォー）
→ 年齢・雇用形態・悩み・転職を考えたきっかけ
→ 読者との共感ポイントを作る

H2：転職活動で実際にやったこと
→ ステップ形式で紹介
→ 使ったエージェント・かかった期間

H2：失敗したこと・大変だったこと
→ 正直に書く（E-E-A-T対策・信頼感）

H2：転職後の変化（アフター）
→ 年収・環境・気持ちの変化

H2：この体験談から学べること
→ 読者へのメッセージ・再現性

[LINE誘導ブロック①：体験談の後]

H2：同じ状況の方へ
→ 「あなたも相談できます」の流れを作る

[LINE誘導ブロック②（記事末尾）]
```

## テンプレートT04：悩み解決記事

**用途：** 読者の具体的な悩みに答える記事
**検索意図：** 「フリーター 30歳 手遅れ」「空白期間 転職 不利」
**CVRが高い理由：** 悩みが深い＝相談への転換率が高い

### 構成テンプレート

```
[タイトル]
フリーター30歳はもう手遅れ？【正直に解説します】

[冒頭（200字以内）]
「30歳のフリーターはもう正社員になれない？」
この記事ではその不安に正直に答えます。
結論を先に言うと：手遅れではありません。ただし、動き方が重要です。

H2：結論：〇〇です（結論ファースト）
→ 「〇〇の悩みは正直どうなのか」に最短で答える

H2：なぜそう言えるのか（根拠）
→ データ・エージェント情報・事例等で根拠を示す

H2：ただし、注意すること
→ 条件・難しい部分を正直に伝える（信頼感UP）

H2：具体的にどうすればいいか
→ ステップ形式で解決策を提示

[LINE誘導ブロック①]

H2：うまくいく人・いかない人の違い
→ 成功パターンと失敗パターンを整理

H2：よくある質問（FAQ）

[LINE誘導ブロック②（記事末尾）]
```

## テンプレートT05：Q&A記事

**用途：** よくある質問を1記事にまとめる・またはQ1個に深掘りで答える
**検索意図：** 「第二新卒 何歳まで」「転職エージェント 費用」
**CVRが高い理由：** 疑問解消 → 信頼構築 → LINE登録の流れが自然

### 構成テンプレート

```
[タイトル]
第二新卒についてよくある質問13個に答えます【2026年版】

[冒頭（200字以内）]
第二新卒の転職に関して「これって聞いていい？」という質問を
13個まとめて正直に答えます。

H2：Q1〜Q3：第二新卒の定義・基本
Q. 第二新卒って何歳まで？
A. [回答]（knowledge/faq.md の回答を参照・転用）

Q. 第二新卒と既卒の違いは？
A. [回答]

Q. 第二新卒枠は有利？不利？
A. [回答]

H2：Q4〜Q7：転職活動の方法
（同形式で続ける）

[LINE誘導ブロック①：H2の途中に挿入]

H2：Q8〜Q11：エージェント・サービス
（同形式で続ける）

H2：Q12〜Q13：不安・メンタル
（同形式で続ける）

[LINE誘導ブロック②（記事末尾）]
```
