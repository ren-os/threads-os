# categories.md — カテゴリ設計・記事マップ

ブログのカテゴリ構造を定義する。
目的は「検索ユーザーをLINE登録・ASP成約まで導くSEO導線を作ること」。

---

## カテゴリ設計方針

- **ターゲット：** 18〜34歳の第二新卒・フリーター・既卒・派遣
- **ゴール：** 記事 → LINE登録 → フォーム回答 → 相談予約 → ASP成約
- **構造：** 大カテゴリ3本を軸に、中カテゴリ・記事を展開する
- **SEO方針：** 大カテゴリ = 検索ボリュームが大きいキーワード。中カテゴリ = 購買意向が高いキーワード

---
