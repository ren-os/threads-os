# internal_links.md — 内部リンク設計

記事間の導線・LINE誘導ポイント・関連記事設計を定義する。
目的は「読者をブログ内で循環させ、最終的にLINE登録まで導くこと」。

---

## 内部リンクの設計方針

- 1記事につき内部リンクを3本以上入れる
- リンクアンカーテキストはリンク先の内容がわかる自然な文章にする（「こちら」「詳細」はNG）
- 読者の回遊を促すリンクと、LINE登録への誘導リンクの2種類を使い分ける
- カテゴリページ（HUBページ）を最も強化し、そこから各記事へリンクを張る

---

## リンク構造の全体設計

```
[カテゴリTOP（HUBページ）]
    ↓ 各記事へのリンク
[個別記事（詳細ページ）]
    ↓ 関連記事へのリンク（横の移動）
    ↓ カテゴリTOPへのリンク（上への移動）
    ↓ LINE登録ページ or フォームへのリンク（コンバージョン）
```

## LINE誘導リンクの設計

### 挿入位置のルール

| 位置 | タイミング | タイプ |
|-----|---------|-------|
| 本文中①（H2の2〜3個目のあと） | 読者が「この人の話を聞いてみたい」と感じ始めたころ | タイプA（軽い誘導） |
| 本文中②（体験談・FAQ直後） | 「自分の場合はどうだろう？」という疑問が生まれたころ | タイプA |
| 記事末尾 | 記事を読み終えて行動意欲が高まったころ | タイプB（強い誘導） |

### LINE誘導リンクのアンカーテキスト例

```
【タイプA：軽い誘導】
「あなたの状況に合った方法を一緒に考えます。
 まずはLINEで話しかけてみてください👉 [LINE登録] ← TODO」

【タイプB：強い誘導（記事末尾）】
「この記事を読んでも迷ったら、LINEで直接相談してください。
 第二新卒・フリーター・既卒の転職相談を無料で受け付けています。
 👉 れんのLINEに登録する ← TODO」
```

## 関連記事ブロックの設計

記事末尾に「関連記事」セクションを設置する。

### 表示ルール

- 関連記事は2〜3本まで（多すぎると読まれない）
- 同じカテゴリの記事を優先する
- 最もCVRが高い記事（エージェントレビュー・比較）を必ず1本入れる

### 関連記事ブロックのテキスト例

```
---
【関連記事】
▶ 第二新卒におすすめの転職エージェント比較5選
▶ 就職カレッジの評判・口コミ【実際に使った感想】
▶ 第二新卒の転職方法【完全ガイド】
---
```

## パンくずリストの設計

全記事に以下の形式でパンくずを設定する（SEO・UX向上）。

```
ホーム > カテゴリ名 > 記事タイトル
例：ホーム > 第二新卒転職 > 第二新卒エージェント おすすめ5選
```

---

## 内部リンクの優先度ルール

| 状況 | 対応 |
|-----|------|
| 新記事を公開した | 既存の関連記事から新記事へのリンクを追加する |
| 記事のPVが高い | その記事からLINE誘導リンクを確実に入れる |
| エージェントレビュー記事 | 比較記事・カテゴリTOPから必ずリンクを張る |
| 体験談記事 | 悩み解決記事・エージェント比較へのリンクを入れる |
