# viral_archive.md — バズ投稿データベース

SNS4プラットフォームのバズ投稿を蓄積・分析する。
投稿生成時は必ずこのファイルを参照し、バズの構造を反映する。

---

## 使い方

```
1. daily_research.md の手順でリサーチ中にバズ投稿を発見したら記録する
2. 週次レビュー時に「構成分析」「CTA分析」を追記する
3. 投稿生成前（generate_posts.md Step1）にこのファイルを読んで
   バズパターンを今回の生成に反映する
```

## research/ → knowledge/ 反映ルール

```
月に3件以上同じパターンのバズが観測された場合：
→ threads/viral_patterns.md に「観測済みバズパターン」として追記する

新しいフックが高パフォーマンスだった場合：
→ knowledge/hooks.md の「未使用フックストック」に追加する

プラットフォームをまたいで同じネタがバズった場合：
→ 最優先でThreadsに転用投稿を作る（threads/ideas.md に追加）
```
