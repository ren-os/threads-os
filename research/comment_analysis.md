# comment_analysis.md — コメント分析・悩み抽出

SNS4プラットフォームのコメント欄から「ユーザーの生の悩み・質問」を抽出し、
投稿ネタ・LINE相談・ASP案件選定に活用する。

---

## 目的

```
コメント欄は「ターゲットの本音の言葉」が集まる最良のリサーチ場所。
→ 「怖い」「わからない」「同じです」等の言葉を拾う
→ その言葉をそのままフック・本文・返信テンプレートに使う
→ 「自分のことを言ってる」と思わせる投稿を作る
```

---

## コメント収集手順

### 毎日（5分）

```
Step1：自分の投稿コメントを確認する
→ 今日の投稿（5本）のコメント欄を全件確認する
→ 質問・悩み・「同じです」系のコメントを記録する

Step2：バズっている競合投稿のコメントを確認する
→ viral_archive.md に記録したバズ投稿のコメントを5〜10件確認する
→ 多くの人が「いいね」しているコメントを優先して記録する

記録先：このファイルの「コメントログ」に追記する
```

### 週次（5分・月曜）

```
Step1：週間コメントを集計する
→ 同じ悩みワードが2回以上出ていないか確認する
→ 「多い悩み TOP3」を抽出する

Step2：投稿ネタに変換する
→ threads/ideas.md に「コメント由来ネタ」として追加する
→ knowledge/pain_points.md（TODO）に新しい悩みワードを追記する
→ line/faq.md によく来た質問を追記する（3回以上来たもの）
```

## 悩み分類システム

コメントを収集したら以下のカテゴリに分類する。

### カテゴリA：行動できない系

```
代表的なコメントワード：
・「何から始めればいいかわからない」
・「一歩が踏み出せない」
・「転職したいけど動けない」
・「怖くて応募できない」
・「準備ができてから動こうと思ってる」

→ 投稿化方針：T06共感型 or T01ストーリー型（同じ状況からの逆転）
→ ネタ候補：「動けない人がまず1つだけやること」
```

### カテゴリB：自己否定系

```
代表的なコメントワード：
・「自分なんか無理」
・「職歴がアルバイトしかない」
・「学歴がない」
・「もう手遅れかな」
・「どうせ落ちる」

→ 投稿化方針：T01ストーリー型（同じ条件からの成功談）
→ ネタ候補：「フリーター期間○年でも通った理由」
```

### カテゴリC：知識不足系

```
代表的なコメントワード：
・「第二新卒って何歳まで？」
・「エージェントって怖くないですか？」
・「履歴書の書き方がわからない」
・「面接で何を聞かれるの？」
・「空白期間はバレる？」

→ 投稿化方針：T05FAQ型 or T07知らないと損するシリーズ
→ ネタ候補：「よく聞かれること、まとめて答えます」
```

### カテゴリD：不安・恐怖系

```
代表的なコメントワード：
・「面接が怖い」
・「落ちたらどうしよう」
・「また同じ職場に入ったらどうしよう」
・「転職して失敗したら」
・「ブラック企業にまた入ったら」

→ 投稿化方針：T06共感型 or T03NG vs OK型
→ ネタ候補：「ブラック企業を見分ける方法」
```

### カテゴリE：具体的質問系

```
代表的なコメントワード：
・「退職理由はどう言えばいい？」
・「履歴書の空白期間はどう書く？」
・「エージェントは何社登録すればいい？」
・「面接で逆質問は何を聞けば？」

→ 投稿化方針：T05FAQ型 or T03NG vs OK型
→ LINEフォロー：相談招待の返信テンプレートを使う
```

### カテゴリF：共感・応援系

```
代表的なコメントワード：
・「同じです！」
・「私もそうでした」
・「頑張ります」
・「背中押してもらいました」

→ 投稿化方針：更に共感系を増やすシグナル
→ ネタ候補：この投稿テーマを別視点で深掘りする
```

## コメント由来ネタ変換ルール

```
変換の基本ルール：

①「ユーザーの言葉」をそのままフックに使う
  例：「転職したいけど動けない」→ フック：「転職したいのに、動けない。」

②「同じ質問が3回来たら即投稿化」
  例：「第二新卒って何歳まで？」→ T05FAQ型の投稿を作る

③「深い共感コメントが来た投稿のテーマは深掘りする」
  例：「私も同じでした」が10件→ 同テーマを別角度で続編投稿

④変換したネタは threads/ideas.md に追記する
  タグを付ける：[コメント由来]
```

## LINE・ASPへの連携ルール

```
コメント内容からわかること → 活用先：

「費用かかりますか？」が多い
→ line/faq.md に追記・投稿ネタ化

「自分みたいな経歴でも相談できますか？」が多い
→ asp/faq.md に追記・objections.md に対処法追加

「エージェントって強引ですか？」が多い
→ knowledge/objections.md の反論4を強化

「○○エージェントってどうですか？」が来たら
→ asp/comparison.md の該当案件情報を確認・回答に活用
```
