# competitor_list.md — 競合・参考アカウント管理リスト

REN OS が毎朝リサーチする競合・参考アカウントのマスターリスト。
daily_research.md と competitor_tracker.md はこのファイルを参照する。

---

## 使い方

```
1. 毎朝 08:30 のリサーチ時にこのファイルを開く
2. 各アカウントの最新投稿を確認する
3. バズ投稿があれば competitor_tracker.md の「伸びた投稿ログ」に記録する
4. アカウント情報が変化したら「最終確認日」と「フォロワー数」を更新する
5. 新しいアカウントを追加する場合は下部のテンプレートをコピーして追記する
```

## 管理ルール

```
追加基準：
・転職・第二新卒・フリーター・既卒・キャリア系のジャンル
・フォロワー数1,000以上（参考アカウントは500以上でも可）
・体験談・ノウハウ・相談系のどれかに該当する

確認頻度の目安：
・毎日：Threadsのメイン競合（TH-01・02）のみ
・週次：その他のThreads・X・TikTok・Instagram・YouTube Shorts
・月次：動きの少ないアカウント・YouTube Shorts の一部

フォロワー数の更新：
・月次（最終月曜）に一覧表を更新する
・急増（1週間で+10%以上）があれば即時更新して要因を分析する

最終確認日の更新：
・確認するたびに必ず更新する
・2週間以上確認できていないアカウントは確認頻度を「月次」に下げる
```
