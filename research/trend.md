# trend.md — トレンド追跡

転職・第二新卒・フリーター・SNSアルゴリズムの最新トレンドを毎日追い、
投稿テーマ・フック・ネタに反映させる。

---

## 追うべきトレンドテーマ

### テーマ1：転職市場

```
毎日確認するポイント：
・転職に関するニュース（企業の大量採用・早期退職・リストラ等）
・求人倍率・雇用統計の変化
・「今転職するなら」という文脈の話題

確認ソース：
□ Yahooニュース「転職」で検索
□ X の「転職」トレンド確認
□ 月次：厚生労働省の雇用統計（リリース日：毎月中旬）

投稿への反映：
→ ニュースを起点に「今が動き時の理由」投稿を作れないか確認
→ trend.md の「転職市場ログ」に記録
```

### テーマ2：第二新卒

```
毎日確認するポイント：
・「第二新卒」の検索量・話題量
・「第二新卒 〇〇」の複合キーワード（Xで確認）
・第二新卒採用を発表した企業情報

確認ソース：
□ Threads：「第二新卒」で検索
□ X：「第二新卒」で最新タブ確認
□ 週次：Google トレンド「第二新卒」（比較：「フリーター転職」）

投稿への反映：
→ 新しい悩みワードが出ていたらフックに転用
→ 旬の話題がある場合は翌日の投稿テーマに組み込む
```

### テーマ3：フリーター・既卒

```
毎日確認するポイント：
・「フリーター」「既卒」のSNS上の文脈
・「フリーターは無理」「既卒は不利」等のネガティブ言説（反論投稿のネタになる）
・フリーター・既卒に関する企業発表・メディア記事

確認ソース：
□ X：「フリーター 正社員」「既卒 転職」で最新タブ
□ TikTok：「フリーター転職」「既卒就活」タグ

投稿への反映：
→ 「無理」「不利」という言説が多い時期は逆転ストーリー投稿を増やす
→ 季節性（卒業シーズン・4月等）をネタに使う
```

### テーマ4：SNSアルゴリズム

```
月次確認するポイント：
・Threadsのアルゴリズム変更情報
・Xのアルゴリズム変更（投稿の伸び方に影響）
・TikTok・Instagramのアルゴリズム変更

確認ソース：
□ Threads公式ブログ・Meta公式発表（月次）
□ SNSマーケティング系のX/Threadsアカウント
□ 「Threadsアルゴリズム 2026」等のキーワード検索

運用への反映：
→ threads/viral_patterns.md の「Threadsのアルゴリズム特性」を更新する
→ automation/daily.md の投稿時間帯を調整する
```

### テーマ5：転職関連ワード（季節性）

```
季節別のホットワード：

3〜4月：卒業・新社会人・新年度・入社
4〜5月：ゴールデンウィーク明け・5月病・新社会人つらい
6〜7月：夏のボーナス後退職・転職活動開始
9〜10月：秋採用・中途採用ピーク
11〜12月：年末年収・年収アップ転職
1〜2月：新年の決意・転職活動計画

→ 季節のホットワードをフックに使う
→ 投稿ネタの仕込みは1ヶ月前から始める
```

## トレンド × 投稿テーマ転換表

リサーチしたトレンドを投稿テーマに変換する際の参考。

| トレンドの発見 | 投稿テーマ候補 | 型 | 緊急度 |
|-----------|------------|---|-------|
| 「第二新卒 不安」が急増 | 「第二新卒の不安、全部答えます」 | T05 FAQ | 高 |
| 企業の早期退職ニュース | 「早期退職後の転職、どう動く？」 | T01 | 中 |
| 卒業シーズン（3〜4月） | 「既卒1年目が一番動きやすい理由」 | T01 | 高（季節） |
| 「エージェント怖い」急増 | 「エージェント怖くないですか？答えます」 | T05 | 高 |
| 5月病ツイート急増 | 「会社行きたくない、その気持ちわかります」 | T06 | 高（季節） |

## research/ → knowledge/ 反映ルール

```
月次トレンドで「3回以上観測されたキーワード」は：
→ knowledge/posting_rules.md のNGワード・使えるワードを更新する
→ threads/hooks.md の未使用ストックに追加する

SNSアルゴリズム変化が確認された場合：
→ threads/viral_patterns.md の「Threadsのアルゴリズム特性（2026年版）」を更新する
→ automation/scheduler.md の投稿時間帯を見直す

季節性ワードの事前仕込み：
→ 該当月の1ヶ月前に threads/ideas.md に季節ネタを追加する
```
