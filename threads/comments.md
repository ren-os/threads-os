# comments.md — コメント返信案生成

Threadsに来たコメントへの返信案を自動生成するためのプロンプト定義書。
返信はエンゲージメントを高め、プロフィール遷移・LINE登録を自然に促す。

---

## コメント返信の基本方針

- 返信は必ず24時間以内に行う
- 感謝・共感を先に伝える
- 回答は短く・わかりやすく
- 自然な流れでLINE誘導（無理に押し込まない）
- NG：論争・感情的な返答・否定

---

## コメントタイプ別返信テンプレート

### タイプ1：共感コメント
「自分もそうです」「わかります」系

```
【返信テンプレート】
「[共感の言葉]、ありがとうございます。
同じ状況の方が多いんだなと感じています。
もし何かお役に立てることがあれば、
プロフィールからお気軽にどうぞ😊」
```

**ルール**
- 相手の言葉を受けて「そうですよね」から入る
- 押し売りせず、自然に案内する

---

### タイプ2：質問コメント
「〇〇はどうすればいいですか？」系

```
【返信テンプレート - 短答できる場合】
「ご質問ありがとうございます！
[答えを1〜2行で簡潔に]。
詳しい話はプロフィールのLINEでもできますので、
よければどうぞ😊」

【返信テンプレート - 詳細が必要な場合】
「ご質問ありがとうございます！
[状況によって変わる部分がある旨を伝える]。
LINEでもう少し詳しく聞かせていただければ
より具体的にお答えできます。
プロフィールからどうぞ😊」
```

**ルール**
- 質問に必ず答えてからCTAを入れる
- CTAなしで答えきれる内容はそのまま答える（信頼構築優先）

---

### タイプ3：批判・否定コメント
「そんなに簡単じゃない」「嘘くさい」系

```
【返信テンプレート】
「おっしゃる通り、全員に当てはまるわけではないです。
[相手の意見を認める一言]。
投稿は一つの例として参考にしていただければ幸いです。」
```

**ルール**
- 言い返さない
- 相手の意見を認めた上で、立場を説明する
- それ以上の議論はしない

---

### タイプ4：体験談シェアコメント
「自分もこうだった」系

```
【返信テンプレート】
「体験を共有してくださりありがとうございます！
[相手の体験への共感を1行]。
そういう経験をした方のお役に立てるよう
引き続き発信していきます😊」
```

**ルール**
- 相手の体験を否定しない
- 次の投稿への期待感につなげる

---

### タイプ5：相談コメント
「〇〇な状況なんですが〜」系

```
【返信テンプレート】
「コメントありがとうございます。
[状況への共感を1行]。
詳しい状況をお聞きできれば、
もう少し具体的なアドバイスができます。
プロフィールのLINEから気軽にどうぞ😊」
```

**ルール**
- コメント欄での詳細相談には限界がある旨を自然に伝える
- LINEへの誘導は「詳しく聞けるから」という理由で自然に行う

---

## コメント返信生成プロンプト

Claudeに返信案を生成させる時のプロンプト：

```
以下のThreadsへのコメントに対する返信案を3パターン作成してください。

【投稿内容】
[投稿本文]

【コメント内容】
[コメントの内容]

【返信のルール】
- 50文字以内（短く・自然に）
- 共感から入る
- 自然な場合のみLINEに誘導する
- 押し売りしない
- 絵文字は0〜1個まで
```
