# ideas.md — 投稿ネタバンク

投稿のネタを優先度・ジャンル別に管理する。
投稿生成前に参照し、ネタを選ぶ。使ったネタは「使用済み」に移動する。

---

## 使い方

1. 生成前にこのファイルを開く
2. 優先度「高」のネタから使う
3. 使ったネタは「使用済み」セクションに移動する
4. ネタが10本を下回ったら補充する

---

## ネタ出しプロンプト（補充時に使う）

ネタが不足してきたら、以下のプロンプトで補充する：

```
以下の条件で投稿ネタを20個出してください。

【ターゲット】
18〜34歳の第二新卒・既卒・フリーター

【使用済みテーマ（避けること）】
threads/posted_log.md の使用済みテーマ一覧を参照

【投稿比率】
ストーリー10本・ノウハウ6本・共感4本

【出力フォーマット】
- [ ] 「フック案」
  → 型・ジャンル・ターゲット

改善案があれば追加でコメントしてください。
```
