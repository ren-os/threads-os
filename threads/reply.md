# reply.md — 返信テンプレート集

Threadsのコメント・DM、LINEへの返信テンプレートを管理する。
コメントの自動生成プロンプトは `threads/comments.md` を参照すること。

---

## Threads コメント返信テンプレート

### 汎用返信（短答用）

```
A: ありがとうございます！参考になれば嬉しいです😊
B: 共感してもらえて嬉しいです。引き続き発信していきます！
C: ご質問ありがとうございます。[答え]。お役に立てれば！
D: おっしゃる通りです。[共感]。ありがとうございます。
```

---

### 相談誘導返信

```
「詳しい状況を聞かせていただければ、もう少し具体的にお伝えできます。
プロフィールのLINEからお気軽にどうぞ😊」

「コメントありがとうございます。
個別の状況に合わせたアドバイスはLINEで対応しています。
よければプロフィールからどうぞ。」
```

---

### 感謝コメントへの返信

```
「コメントありがとうございます！励みになります。
引き続き役立つ情報を発信していきますね。」

「そう言っていただけると嬉しいです。
何かあればいつでも気軽にどうぞ😊」
```

### 状況確認後の返信

```
【フリーターの場合】
「フリーターからの転職、
今の状況をもう少し詳しく教えてください。
・年齢
・バイト経験の内容
・転職したい時期
・希望の職種（あれば）
これを教えていただければ具体的なアドバイスができます。」

【既卒の場合】
「既卒からの転職活動、
一緒に整理していきましょう。
まず聞かせてください。
・卒業後どのくらい経ちますか？
・今まで就活はしましたか？
・希望の業種はありますか？」

【第二新卒の場合】
「第二新卒での転職を考えているんですね。
整理しましょう。
・前職を辞めた（辞める）理由は？
・希望する職種・業種はありますか？
・転職の希望時期は？」
```

## 返信のNGパターン

| NG | 理由 |
|----|------|
| 長文で説明しすぎる | 読まれない・重さを感じさせる |
| 「今すぐ登録してください」 | 圧力になる |
| 返信を24時間以上放置する | 信頼を失う |
| コメント欄で詳細な相談に乗る | LINEへの誘導機会を失う |
| 批判コメントに感情的に返す | 炎上リスク |
