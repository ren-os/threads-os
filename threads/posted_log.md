## 記録フォーマット

```
## セット番号：[N] / 投稿日：YYYY-MM-DD

| ID | 型 | ジャンル | フック冒頭（20文字） | 投稿時刻 | 閲覧数 | いいね | 保存 | 返信 | プロフ遷移 | 評価 |
|----|----|---------|--------------------|---------|--------|-------|------|------|-----------|------|
| 1  | T01 | フリーター | フック | 07:00 | - | - | - | - | - | - |
```

**評価基準**
- A：閲覧数が平均の1.5倍以上 かつ いいね率1%以上
- B：平均水準
- C：閲覧数が平均の50%以下 または いいね率0.3%未満
- S：閲覧数が平均の2倍以上 かつ 保存率0.3%以上
