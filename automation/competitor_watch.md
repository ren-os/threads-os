## 週次競合チェック手順（毎週月曜・15分）

```
Step1：Threadsアカウント確認（各3分・計9分）
→ C01・C02・C03 を順番に確認する
→ 直近1週間の投稿を全件スクロールする
→ 以下を確認する：
   □ 直近でバズっている投稿はあるか（いいね100以上目安）
   □ 新しい投稿の型・フック・ジャンルは何か
   □ CTA・LINE誘導の方法に変化があるか

Step2：発見の記録（3分）
→ バズ投稿があれば analysis/competitors.md のログに記録する
→ 転用できるアイデアがあれば threads/ideas.md に追加する

Step3：れんへの転用判断（3分）
→ 「この週に試せる転用アイデア」を1つ決める
→ 翌週の投稿生成時に反映する
```

---

## 月次競合深掘りチェック（毎月最終週・30分）

```
Step1：Threads全体の傾向確認（10分）
→ C01〜C03 の1ヶ月分のパフォーマンスを確認する
→ 月間バズ投稿 TOP3 を特定する

Step2：ブログ競合確認（10分）
→ C04・C05 の新着記事を確認する
→ 上位表示されている記事のタイトル・構成を確認する
→ 自分のブログに不足しているテーマがないか確認する

Step3：プロフィール・導線確認（5分）
→ 競合のプロフィール文・CTA・LINE誘導方法に変化があれば記録する

Step4：analysis/competitors.md の月次分析テンプレートを記入（5分）
```

## バズパターン転用ルール

競合のバズ投稿を見つけた場合、そのままコピーしない。
以下のルールで「型・テーマ・フック」を参考にしながら独自の内容に変換する。

```
転用ルール：
1. バズの「型」を参考にする（T01〜T10）
2. バズの「テーマ」を参考にする（同じ悩みでれんの言葉で書く）
3. 「フック」はそのままコピーしない（言い換える）
4. 転用した場合は analysis/competitors.md の「転用済み」にチェック
```
