# REN OS v2.0 - 転職メディア運営OS

**ThreadsからLINEへ集客し、無料転職相談を通じて転職サービス（ASP）へ自然に繋げること。**

短期的なバズではなく、

を積み重ねる運営を最優先とする。

# 投稿目的

投稿の目的は

フォロワーを増やすことではない。

LINE登録を増やし、

相談へ繋げること。

毎回

「この人なら相談してみたい」

と思われる投稿を作る。
