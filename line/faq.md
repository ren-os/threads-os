# faq.md — LINE用FAQ

LINEユーザーからよく来る質問とその回答テンプレート。
reply_templates.md と連動して使う。

---

## FAQ運用ルール

- この文書は「回答のベース」として使う。実際の返信は状況に合わせて言葉を調整すること
- 新しい質問が3回以上来たら追加する
- 回答が変化したら随時更新する

## カテゴリ1：費用・安全性

### Q1. 費用はかかりますか？

**回答：**
```
完全無料です。

転職エージェントの紹介含め
求職者側に費用は一切かかりません。

費用が発生することはないので安心してください。
```

---

### Q2. 強引な勧誘はありますか？

**回答：**
```
しません。

「今すぐ転職して」「このサービス使って」
という勧誘は一切しません。

話すだけでもOKです。
断っても全然大丈夫です。
```

---

### Q3. 個人情報は安全ですか？

**回答：**
```
フォームに記入いただいた情報は
相談の準備にだけ使います。

第三者への提供や
転職エージェント以外への共有はしません。

※ 転職エージェントへの情報共有は
　 ご本人の同意を取ったうえで行います。
```

### Q4. フリーターでも正社員になれますか？

**回答：**
```
なれます。

年齢と動き方によって難易度は変わりますが
20代のうちに動けば選択肢は多いです。
```

### Q5. 既卒・空白期間があっても大丈夫ですか？

**回答：**
```
大丈夫です。

確かに「空白期間」は面接で聞かれますが
伝え方次第で印象はかなり変わります。

一緒に「どう伝えるか」を整理できます。

今どんな状況か教えてもらえますか？
```

---

### Q6. 学歴が低くても転職できますか？

**回答：**
```
できます。

学歴よりも「何をしてきたか」「何ができるか」を
見る企業は多いです。

ただ、業界や職種によっては
学歴が影響することもあるので
狙う方向性によって変わります。

状況を聞かせてもらえれば
あなたに合った方向性を一緒に考えます。
```

---

### Q7. 第二新卒って何歳までですか？

**回答：**
```
一般的には「卒業後3年以内」が目安ですが
企業によって定義が違います。

25〜26歳であれば第二新卒として
見てもらえる会社は多いです。

あなたは今何歳ですか？
```

---

### Q8. 未経験の職種に転職できますか？

**回答：**
```
できます。

ただ、職種によって難易度が変わります。

未経験OKの求人が多い職種：
・IT/Web系（特にエンジニア・営業）
・営業
・事務

あなたがどんな職種を考えているか
教えてもらえれば、もう少し詳しくお伝えできます。
```

## カテゴリ4：転職活動・エージェント

### Q13. 転職エージェントとはどんなサービスですか？

**回答：**
```
転職エージェントは
「求人紹介 + 転職サポート」を無料でしてくれるサービスです。

・求人紹介
・履歴書・職務経歴書の添削
・面接対策
・企業との交渉

これを求職者側の費用0円でやってもらえます。

相談の中でどんなサービスが合うか
一緒に確認できます。
```

---

### Q14. 転職サイトと転職エージェントの違いは何ですか？

**回答：**
```
簡単に言うと…

転職サイト → 自分で求人を探して応募する
転職エージェント → 担当者が一緒に動いてくれる

エージェントはサポートが手厚い分
担当者との相性が大事になります。

どちらが合うか一緒に考えられますよ。
```

---

### Q15. 複数の転職エージェントに登録してもいいですか？

**回答：**
```
OKです。

むしろ2〜3社を比較しながら使う人が多いです。
担当者との相性もあるので
並行して使うのが一般的です。

ただ管理が大変になるので
最初は1〜2社から始めるのがおすすめです。
```
