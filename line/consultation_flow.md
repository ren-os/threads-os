# consultation_flow.md — 相談〜内定までのフルフロー

LINE登録から内定・ASP報酬獲得までの全工程を定義する。
Threads → LINE → 相談 → 求人紹介 → 応募 → 内定 → ASP報酬 の順番。

---

## フロー全体図

```
[Threads投稿]
    ↓ プロフィール遷移
[LINE登録]
    ↓ greeting.md（挨拶・フォーム誘導）
[Googleフォーム回答]
    ↓ reply_templates.md（日程調整）
[個別相談]
    ↓ 転職方向性の整理
[転職エージェント紹介・登録]
    ↓ エージェント担当者による求人紹介
[求人応募]
    ↓ 書類選考 → 面接
[内定]
    ↓ 入社
[ASP成果確定・報酬獲得]
```

---

## フェーズ1：LINE登録〜フォーム回答

### 目的
- ユーザーの現状把握
- 相談への移行

### アクション

| ステップ | 内容 | 担当 |
|---------|------|------|
| 登録直後 | greeting.md の3メッセージを自動送信 | LINE自動 |
| 登録後48時間 | フォーム未回答の場合フォロー | LINE自動 |
| フォーム回答後 | reply_templates.md「フォーム回答後の最初の返信」を送る | 手動 |

### Googleフォーム収集項目（TODO：フォーム作成後に更新）

- 名前（ニックネームでもOK）
- 年齢
- 雇用形態（正社員・フリーター・既卒・第二新卒・派遣・その他）
- 今の悩み・状況（自由記述）
- 転職を考え始めたきっかけ
- 転職の希望時期（1ヶ月以内 / 3ヶ月以内 / 半年以内 / まだ決めていない）
- 希望の相談形式（TODO: 設定次第で変更）

## フェーズ2：個別相談

### 目的
- ユーザーの状況・ゴールを明確化する
- 信頼関係を構築する
- 次のアクション（転職エージェント登録）を合意する

### 相談の流れ（30〜60分）

**① 状況ヒアリング（15分）**

```
確認事項：
・現在の雇用形態・職歴
・今の悩み・転職を考えた理由
・転職で叶えたいこと（収入UP / 安定 / やりがい / 環境改善）
・転職の希望時期
・過去に転職活動をしたことがあるか
```

**② 転職方向性の整理（15分）**

```
一緒に整理すること：
・ターゲット職種・業界
・応募戦略（第二新卒枠 / 既卒 / 未経験 / 経験者転職）
・優先順位（給与 / 環境 / やりがい / 勤務地）
・強みの言語化（バイト経験・学校経験含む）
```

**③ 次のアクション提案（15分）**

```
提案する内容：
・おすすめの転職エージェント（asp.mdを参照）
・エージェント登録の手順
・書類作成・面接対策のスケジュール感
・次回連絡のタイミング
```

### 相談後にすること

```
① 相談後フォローアップを送る（reply_templates.md参照）
② 紹介した転職エージェントへの登録を確認する
③ LINEで進捗フォローを続ける
```

| 未経験ITエンジニア | [TODO: 該当ASP確認後に追加] |

## フェーズ4：求人応募〜書類選考

### れんの関わり方
- 主にLINEで進捗確認・励まし
- 書類・面接の相談が来たら answer する
- エージェント担当者がメインで動くため、れんは「伴走者」ポジション

### 書類対策（相談が来た場合）

```
履歴書の自己PR確認ポイント：
・志望動機が「前向きな理由」になっているか
・バイト経験をスキルとして言語化できているか
・空白期間の説明が「これからの話」になっているか

職務経歴書（必要な場合）：
・数字を入れる（売上〇%UP / 〇人対応 等）
・業務フローではなく「成果・貢献」を書く
```

### 面接対策（相談が来た場合）

```
よく聞かれること：
① 自己紹介（1〜2分）
② 志望動機
③ 退職理由・空白期間の説明
④ 自分の強み・弱み
⑤ 入社後にやりたいこと

NG例 → OK例 は knowledge/objections.md を参照
```

## フェーズ5：内定〜入社

### れんの関わり方
- 内定が出たら「おめでとう」メッセージを送る
- 内定承諾・辞退の判断を相談された場合は一緒に整理する
- 入社後の進捗を定期的に確認する（感謝の体験談を後日シェアしてもらえるよう関係維持）

### 内定後のメッセージ例

```
内定おめでとうございます🎉

一緒に動いてきた甲斐がありましたね。

入社に向けて何か不安なことや
確認したいことがあれば
いつでもメッセージしてください。

引き続きサポートします😊
```
