# step_messages.md — 登録後7日間のステップ配信

友達追加後に自動送信されるメッセージ設計。
「教育 → 信頼構築 → 相談誘導」の順番で設計する。

---

## 設計方針

- 1日1通（情報過多でブロックを防ぐ）
- 売り込みは7日目まで入れない
- Day1〜5で信頼を積み上げ、Day6〜7で相談を誘導する
- 各メッセージは150〜250文字程度（読みやすさ優先）

---

## 配信スケジュール

| Day | 送信タイミング | テーマ | 目的 |
|-----|------------|-------|------|
| 0 | 友達追加直後 | 挨拶・フォーム誘導 | greeting.md に記載 |
| 1 | 翌朝 9:00 | れんの自己紹介 | 「誰に相談するのか」を伝える |
| 2 | 2日後 9:00 | 成功体験談① | 「自分もできる」イメージを作る |
| 3 | 3日後 9:00 | よくある不安の解消 | 「あなたの悩みは普通」を伝える |
| 4 | 4日後 9:00 | 転職ノウハウ | 「有益な情報をくれる人」の印象を作る |
| 5 | 5日後 9:00 | 成功体験談② | 異なるターゲット属性で共感を広げる |
| 6 | 6日後 9:00 | 相談内容の説明 | 「相談って何をするの？」の不安を消す |
| 7 | 7日後 9:00 | 相談誘導・CTA | フォーム回答・相談予約へ転換する |

## Day 3：よくある不安の解消

```
おはようございます、れんです。

よく聞かれる不安を3つ正直に答えます。

Q. 職歴がバイトだけでも大丈夫？
→ 大丈夫です。書き方次第で変わります。

Q. 既卒・空白期間があると不利？
→ 不利な面もありますが、伝え方で印象は変わります。

Q. 転職エージェントって怖くない？
→ 求職者は完全無料で使えます。
　 怖くなる前に一度話を聞いてみてください。

他にも不安なことがあれば
このまま返信してもらえれば答えます😊
```

## Day 4：転職ノウハウ

```
おはようございます、れんです。

今日は一つ知っておいてほしいことを。

面接官が「退職理由」で見ているのは
「うちでも同じ理由で辞めるか」です。

だから正直に言うのはいいのですが
言い方が大事になります。

NG：「人間関係が辛くて」
OK：「チームワークを大切にできる環境を求めて」

内容は同じでも、印象は全然変わります。

面接前にこれを知っているだけで
通過率が変わります。

他にも知りたいことがあれば
いつでも聞いてください😊
```

## 配信後のフォローシナリオ

### フォーム回答済みの場合
→ ステップ配信を一時停止し、個別対応に切り替える

### 7日間で返信・フォーム回答なしの場合
→ 8日目以降は broadcast.md の定期配信に移行する

### 途中でメッセージが来た場合
→ ステップ配信は続けつつ、reply_templates.md で個別返信する
