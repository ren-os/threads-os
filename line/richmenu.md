# richmenu.md — リッチメニュー設計

LINEのリッチメニューはユーザーがトーク画面を開いた時に常時表示されるメニュー。
「次に何をすればいいか」を迷わせない導線設計が最重要。

---

## 設計方針

- ファーストアクションは「無料相談・フォーム」に誘導
- 不安・疑問を解消するボタンを置く
- 操作を簡単にする（タップ1回で完結）

---

## レイアウト構成

```
┌─────────────────────────────────┐
│   大バナー（上部）                 │
│   「転職のお悩み、無料で相談できます」 │
│   ※ 画像 or テキスト              │
└─────────────────────────────────┘
┌───────────┬───────────┬───────────┐
│  ① 無料相談  │  ② よくある │  ③ 転職診断  │
│  を予約する  │    質問    │  を受ける   │
├───────────┼───────────┼───────────┤
│  ④ 体験談   │  ⑤ 自己    │  ⑥ メッセ   │
│  を見る    │  紹介     │  ージを送る  │
└───────────┴───────────┴───────────┘
```

---

## 各ボタンの設計

### ① 無料相談を予約する（最重要）

| 項目 | 内容 |
|------|------|
| 表示テキスト | 無料相談を予約する |
| アクション | Googleフォームを開く |
| リンク | TODO（Googleフォームリンク） |
| 目的 | フォーム回答→相談予約への転換 |
| 配置 | 左上（最も目立つ位置） |

---

### ② よくある質問

| 項目 | 内容 |
|------|------|
| 表示テキスト | よくある質問 |
| アクション | テキストメッセージ送信「よくある質問」 |
| 応答 | 自動返信でFAQリストを送る |
| 目的 | 不安・疑問の解消 |
| 配置 | 中央上 |

**自動返信メッセージ（「よくある質問」受信時）**
```
よくある質問です😊

Q. 費用はかかりますか？
→ 完全無料です。

Q. すぐに転職しなくてもいいですか？
→ 「まだ迷ってる」段階でも大歓迎です。

Q. フリーターでも正社員になれますか？
→ なれます。動き方次第です。

Q. 相談はどんな形式ですか？
→ TODO（DM / ビデオ / 電話のいずれか決定後に記入）

他に質問があれば気軽にメッセージしてください😊
```

### ④ 体験談を見る

| 項目 | 内容 |
|------|------|
| 表示テキスト | 体験談を見る |
| アクション | テキストメッセージ送信「体験談」 |
| 応答 | 自動返信で体験談を1〜2本送る |
| 目的 | 信頼構築・「自分もできる」イメージを持たせる |
| 配置 | 左下 |

**自動返信メッセージ（「体験談」受信時）**
```
相談者の体験談をご紹介します😊

【体験談①】
25歳フリーター→3ヶ月で正社員内定

「バイトしか経験がなくて
もう無理かなと思ってましたが、
書き方と面接を一緒に練習したら
3ヶ月で内定が出ました。
一人で悩んでた時間がもったいなかった」

【体験談②】
既卒1年半→2ヶ月で正社員

「既卒はもう手遅れと思っていたが
戦略を立て直したら意外と早く決まった」

あなたの状況に合った方法を
一緒に考えますよ😊
```

### ⑤ 自己紹介

| 項目 | 内容 |
|------|------|
| 表示テキスト | 自己紹介 |
| アクション | テキストメッセージ送信「自己紹介」 |
| 応答 | 自動返信でれんの自己紹介を送る |
| 目的 | 「誰に相談するのか」の信頼感を醸成する |
| 配置 | 中央下 |

**自動返信メッセージ（「自己紹介」受信時）**
```
改めて自己紹介します。

れんといいます。
第二新卒・既卒・フリーターの方を中心に
転職相談をしています。

[TODO: れんの経歴・相談実績・強みを記入]

「転職を考えているけど不安」
「何から始めればいいかわからない」

そういう段階から一緒に動けます。
気軽にメッセージしてください😊
```

### ⑥ メッセージを送る

| 目的 | 自由なメッセージでの相談窓口を開く |
| 配置 | 右下 |

## リッチメニュー切り替え設計

フェーズによってリッチメニューを切り替える（LINE公式アカウントの機能を使用）。

| フェーズ | 表示メニュー | 切り替え条件 |
|---------|------------|-----------|
| 登録直後 | デフォルトメニュー（上記6ボタン） | 全ユーザー共通 |
| フォーム回答済み | 「相談日程を決める」ボタンを追加 | TODO（タグ管理が必要） |
| 相談完了済み | 「進捗を教える」ボタンを追加 | TODO |
