# greeting.md — 友達追加時の挨拶設計

LINE追加直後に送る最初のメッセージ。
第一印象で「相談してよかった」と思わせる設計。

---

## 設計方針

- 売り込まない
- 「安心」「歓迎」「低ハードル」を最優先に伝える
- 3メッセージに分けて送る（一度に長文を送らない）

---

## メッセージ構成（3通に分割）

### メッセージ1：歓迎＋自己紹介

```
れんです。
友達追加ありがとうございます😊

このLINEでは、
第二新卒・既卒・フリーターの方の
転職相談を無料でお受けしています。

「何から始めればいいかわからない」
「自分みたいな経歴でも大丈夫？」

そういう段階でも全然OKです。
まず話すだけでも大丈夫ですよ。
```

---

### メッセージ2：できること説明

```
このLINEでできること👇

✔ 今の状況を一緒に整理する
✔ 転職活動の進め方をアドバイスする
✔ あなたに合った転職サービスを紹介する
✔ 履歴書・面接の相談に乗る

費用は一切かかりません。
強引な勧誘もしないので安心してください。
```

---

### メッセージ3：フォーム誘導

```
まず、あなたの状況を教えてください。

以下のフォームに答えていただくと
より具体的なアドバイスができます。

👇 30秒で入力できます
[Googleフォームリンク]  ← TODO

「まだ迷ってる」という方も
気軽にメッセージを送ってくださいね😊
```

---

## フォーム未回答者へのフォローメッセージ

登録から48時間後にフォーム未回答の場合に送る。

```
こんにちは、れんです。

先日友達追加していただきましたが
フォームはご覧になりましたか？

「難しそう」「長そう」と思った方、
実は30秒ほどで入力できます。

気軽に答えてみてください😊
👇
[Googleフォームリンク]  ← TODO

もしメッセージで話したい場合は
こちらに直接送っていただいても大丈夫です。
```

---

## 信頼獲得のポイント

| ポイント | メッセージでの表現 |
|--------|----------------|
| 費用への不安 | 「費用は一切かかりません」 |
| 勧誘への警戒 | 「強引な勧誘もしない」 |
| 行動への不安 | 「話すだけでも大丈夫」「まだ迷っててもOK」 |
| ハードルの高さ | 「30秒で入力できます」 |

---

## 運用メモ

- メッセージ1〜3は連続して送る（時間差3秒程度）
- フォーム未回答フォローは step_messages.md のDay2と統合可能
- ブロック率が高い場合は文章を短くする
