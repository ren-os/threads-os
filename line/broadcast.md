# broadcast.md — 定期配信テンプレート

ステップ配信（7日）が終了したユーザーへの継続的な接触設計。
「有益情報を届け続けることで信頼を維持し、行動のタイミングを逃さない」が目的。

---

## 設計方針

- 配信頻度：週1〜2回（多すぎるとブロックされる）
- 売り込みは4回に1回以下
- 有益情報 → 共感 → 相談誘導 のローテーション
- Threads投稿と連動させ、LINE限定感を出す

---

## 週次配信スケジュール

| 曜日 | 送信時刻 | コンテンツタイプ | 目的 |
|------|---------|--------------|------|
| 火曜 | 20:00 | 転職ノウハウ or 体験談 | 有益情報提供・信頼維持 |
| 土曜 | 10:00 | 共感投稿 or 相談誘導 | エンゲージメント・転換 |

---

## テンプレート集

### TYPE A：転職ノウハウ配信

```
こんにちは、れんです😊

今日は一つ知っておいてほしいことを。

[ノウハウの内容 150文字以内]

これを知っているだけで
結果が変わることがあります。

何か気になることがあれば
いつでも返信してください😊
```

**ノウハウネタのソース：**
`threads/ideas.md` の「優先度：高」ネタ（知らないと損するシリーズ等）を転用

**ノウハウ例：**

```
【退職理由の言い換え】

面接で「退職理由」を正直に言うのはOKです。
でも言い方が大事。

NG：「残業が多くて辛くて」
OK：「長く安定して働ける環境を求めて」

内容は同じでも印象は全然変わります。
面接前に必ず言葉を整えておいてください。

気になることがあれば返信どうぞ😊
```

```
【みなし残業の確認方法】

求人票の「みなし残業〇時間含む」は必ず確認を。

例：基本給22万・みなし45時間込み
→ 実質的な基本給は16〜17万円程度

入社前に必ず「実際の平均残業時間」を面接で聞いてください。
曖昧な返答が来たら要注意です。

質問があれば気軽に😊
```

---

### TYPE B：体験談配信

```
こんにちは、れんです😊

相談者の話をさせてください。

[体験談の内容 200文字以内]

同じような状況の方、
気軽にメッセージしてください😊
```

**体験談ネタのソース：**
`threads/posted_log.md` の高評価ストーリー投稿を再編集して使用

**体験談例：**

```
こんにちは、れんです😊

相談者の話をさせてください。

27歳、派遣社員3年目の方でした。
「契約が切れそうで怖い」と相談に来てくれました。

一緒に状況を整理して
正社員向けの求人に絞ったところ
2ヶ月後に正社員として内定が出ました。

「もっと早く動けばよかった」というのが
その方の感想でした。

似た状況の方、返信してください😊
```

### TYPE C：共感配信

```
こんにちは、れんです。

[ターゲットの悩みを代弁する共感文 100文字以内]

その気持ち、全然おかしくないです。

何から始めたらいいか一緒に考えますよ。
気軽にメッセージしてください😊
```

**共感例：**

```
こんにちは、れんです。

「転職したいけど、動けない」

その気持ち、すごくわかります。

何が怖いのか、何がわからないのか
一緒に整理するだけでも全然いいです。

気軽にメッセージしてください😊
```

## Threads投稿との連動ルール

- Threadsで反応が良かった投稿（保存率・返信が多い）をLINE配信に転用する
- Threads投稿の「知らないと損するシリーズ」はLINEノウハウ配信に最適
- 参照先：`threads/viral_patterns.md` の高パフォーマンス投稿ログ
