# objections.md — 反論・不安への対処

ターゲットがLINE登録・相談・転職行動をためらう理由と、それへの対処法をまとめる。
投稿・相談対応・CTAの設計に活用すること。

---

## 反論の構造

```
行動をためらう人は「リスク > メリット」と感じている。
→ メリットを見せ、リスクを下げることで行動を促す。
```

---

## 反論一覧と対処法

### 反論1：「自分なんか無理だと思う」

**背景**
フリーター・既卒・短期離職の経験から自己評価が低くなっている。

**対処法**
- 同じ状況から内定が出た体験談を見せる（→ストーリー投稿）
- 「特別な経験は要らない、書き方・伝え方の問題」と伝える
- 「まずは話を聞くだけでOK」とハードルを下げる

**投稿で使える表現**
```
「自分には無理」と思い込んでいた25歳が
3ヶ月後に内定を取った話をします。
```

---

### 反論2：「転職して失敗したらどうしよう」

**背景**
現状維持バイアス。失敗のイメージが強く動けない。

**対処法**
- 「失敗した後の話」もリアルに投稿する（怖くならない程度に）
- 「情報収集だけなら失敗はない」と伝える
- 「今の状況を続けることもリスク」と気づかせる

**投稿で使える表現**
```
転職で失敗する人の特徴、正直に書きます。
（知っておけば避けられる）
```

---

### 反論3：「何から始めればいいかわからない」

**背景**
情報過多と行動の手順がわからないことで思考停止している。

**対処法**
- 「最初にやること3つだけ」と具体的な手順を見せる
- 「応募しなくてもいい、情報収集から」と伝える
- 相談することで手順を一緒に整理できると伝える

**投稿で使える表現**
```
転職、何から始めたらいいかわからない人へ。
最初にやること、3つだけです。
```

---

### 反論4：「LINEに登録するのが怖い・面倒」

**背景**
個人情報・勧誘・しつこい連絡への不安。

**対処法**
- 「費用ゼロ」「相談するだけでOK」「強引な誘導なし」を明示する
- 「見るだけ・診断だけでもOK」とハードルを下げる
- ブロックできることを自然に伝える

**投稿で使える表現**
```
まず30秒診断だけでもどうぞ。
（強引な勧誘はしません）
```

---

### 反論5：「時間がない・忙しい」

**背景**
今の仕事・バイトに追われて転職を後回しにしている。

**対処法**
- 「転職活動は隙間時間でできる」と伝える
- 「今のまま続けることの時間的コスト」を気づかせる
- 「最初の相談は30分程度でできる」と伝える

**投稿で使える表現**
```
「忙しくて転職活動できない」という人ほど
先に環境を変えた方が早い場合が多い。
```

---

### 反論6：「どうせ受からないだろう」

**背景**
書類・面接での失敗経験から自信を失っている。

**対処法**
- 「落ちた理由は能力ではなく、やり方の問題」と伝える
- 書類・面接の「改善前 vs 改善後」を具体的に見せる
- 「一緒にやれば変わる」という安心感を与える

**投稿で使える表現**
```
書類選考30社落ちた人が
書き方を変えたら次の1社から連絡が来た話。
```

---

### 反論7：「今じゃなくてもいい」（先送り）

**背景**
現状への慣れ・変化への抵抗感。

**対処法**
- 「第二新卒のタイミングは限られている」と伝える（焦らせすぎない）
- 「情報収集だけなら今すぐできる」と伝える
- 「動かなかった後悔」の体験談を見せる

**投稿で使える表現**
```
「いつか動こう」と思い続けて
契約更新を断られる1週間前に気づいた話。
```

---

## 投稿への落とし込み方針

- 反論の存在を認めてから解決策を出す（「その気持ちわかります」から始める）
- 反論を否定しない（「甘えです」はNG）
- 小さな一歩を提示する（「応募しなくていい」「話すだけでOK」）
