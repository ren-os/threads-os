# asp.md — ASP案件管理

相談後に最適な案件を提案するための情報を整理する。
投稿では案件名を基本的に出さない。相談後に紹介する。

---

## 基本方針

- 相談者の状況に最適な案件を選んで提案する
- 案件ありきで相談を誘導しない（信頼喪失につながる）
- 相談者のメリットを最優先にする

## 案件選定フロー（相談者別）

```
相談者の状況を確認
↓
┌── フリーター・職歴なし → 第二新卒エージェントneo / UZUZ
├── 既卒・就活失敗 → 就職カレッジ / 第二新卒エージェントneo
├── 第二新卒（1〜3年離職） → 第二新卒エージェントneo / UZUZ
├── 契約・派遣から正社員へ → キャリアスタート / 第二新卒エージェントneo
└── 未経験転職希望 → キャリアスタート / UZUZ
```
