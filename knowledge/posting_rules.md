# posting_rules.md — 投稿ルール

---

## 基本スペック

| 項目 | ルール |
|------|--------|
| 文字数 | 150〜350文字（本文のみ） |
| 改行 | 1〜2文ごとに改行。詰め込まない |
| 冒頭 | 1行で続きを読みたくなるフックを置く |
| 言葉 | 専門用語禁止。中学生でも読めるレベルで |
| CTA | 10投稿に1〜2本だけ。毎回入れない |
| トーン | 親しみやすい丁寧語。友達の先輩レベル |

---

## 構成テンプレート（基本）

```
【フック】（1行）
↓
【本文】（共感 / 情報 / ストーリー）
↓
【余韻 or CTA】（1〜2行）
```

---

## フックのルール

- 冒頭1行で「続きを読む」を押させる
- 20文字以内が理想
- 数字・具体性・共感ワードを使う

**OK例**
- 「25歳フリーター、内定まで3ヶ月かかった話をします。」
- 「知らないと損する転職の話。」
- 「面接、怖いですよね。」

**NG例**
- 「転職について書きます。」（ぼんやりしすぎ）
- 「こんにちは！今日は〜についてです！」（読まれない）

---

## 本文のルール

- 1段落2〜3行まで。長い文章は分割する
- 箇条書きは5項目まで（多すぎると読まれない）
- 体験談は「状況→転機→結果→学び」の流れで
- 比較は「NG例 → OK例」の順で書く

---

## NGワード・NG表現

| NG | 理由 |
|----|------|
| 「絶対に」「必ず」 | 成果保証になる |
| 「今すぐ」「急いで」 | 煽りになる |
| 「楽して」「簡単に」 | 誇大表現 |
| 「うちに登録しないと損」 | 不安商法 |
| 具体的な会社名・サービス名 | トラブルリスク |
| ブラック企業と断定する表現 | 名誉毀損リスク |
| 「転職すべき」「辞めるべき」 | 強制表現 |

---

## LINE導線ルール（必須）

| ルール | 内容 |
|--------|------|
| URL直貼り | **禁止**。通常投稿の本文にLINE URLを入れない |
| CTA標準フレーズ | **「無料相談はプロフィールのリンクから👇」** で統一 |
| URL管理 | `knowledge/offer.md` の `LINE_URL` のみで管理する |
| プロフィール | LINEリンクを設置済みとする前提で投稿を書く |

---

## CTAのルール

- 10投稿に1〜2本だけ
- 本文で信頼を作った後、最後に自然に入れる
- 標準フレーズ「無料相談はプロフィールのリンクから👇」を使う

**CTA標準フレーズ（これだけ使う）**
```
無料相談はプロフィールのリンクから👇
```

**前置きと組み合わせる場合**
```
何から始めればいいかわからない方は、無料相談はプロフィールのリンクから👇
同じ状況の方は、無料相談はプロフィールのリンクから👇
気になった方は、無料相談はプロフィールのリンクから👇
```

---

## 型の連続使用禁止ルール

- 同じ型（ストーリー・チェックリスト等）を2投稿連続で使わない
- 同じフック冒頭を2週間以内に再利用しない
- 「知らないと損するシリーズ」は1日1本まで

---

## 絵文字のルール

- 使いすぎない（1投稿に0〜2個まで）
- CTA末尾の「😊」「🙌」は自然な範囲でOK
- 本文中の絵文字は原則なし

---

- [ ] 専門用語を使っていないか
- [ ] NGワード・NG表現がないか

- [ ] hooks.md の使用済みフックと被っていないか
